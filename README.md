## 使用仓库相关命令

### 安装 pnpm
```
npm i pnpm -g
```
### 安装所有依赖
```
pnpm install
```
### 启动本地测试项目
```
pnpm run dev
```
### 打包组件库
```
pnpm run build:components
```
### 启动文档
```
pnpm run docs:dev
```
